📦 Server Panel (Tkinter)

A simple Python-based server manager GUI for launching and controlling local server instances (e.g. Minecraft servers).

🚀 Features
Start / stop servers with a GUI
Select server folders from a list
Per-server RAM settings
Live console output inside the app
Command input system
Server log saving system
Light / dark theme toggle
Open server folders instantly
📁 Folder Structure
servers/
  ├── myserver/
  │     ├── server.jar
  │
  ├── another-server/
  │     ├── start.bat
  │
  └── logs/
🧠 Requirements
Python 3.10+
Java (required for .jar servers)
